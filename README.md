# Cozy Castle

Cozy Castle is a cozy life simulation game inspired by the gameplay of Animal Crossing and Stardew Valley, featuring an overworld pixel art style influenced by Pokémon Black & White.

## Overview

The project is currently in active development as a solo project using Godot 4 (C#).

## Technologies

- Godot 4 (C#)
- Aseprite
- Blender
- Git/GitHub
